Sólidos Geométricos

Sólidos geométricos – São objetos tridimensionais (possuem 3 dimensões), que são: largura, comprimento e altura. A largura é a distância de uma ponta de um objeto até a outra ponta, sempre é a menor medida de um objeto. A altura é a dimensão de um objeto, que é a medida da parte mais baixa de um objeto até a parte mais alta deste objeto. O comprimento é a extensão das extremidades de um objeto de uma extremidade até a outra.

Os sólidos geométricos que não se enquadram nas duas categorias anteriores são o que chamamos de outros. Geralmente são sólidos que possuem uma “face” curva, mas que não rolariam se colocados sobre uma superfície plana. Exemplos:

Largura: A medição do lado menor de uma caixa

Comprimento: A ponta de uma madeira até a outra ponta, do lado horizontal.

Altura: Dimensão do chão de uma casa até o teto.
